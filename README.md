# OOP Calculator (C++)

A console-based calculator built using Object-Oriented Programming principles in C++.

## Features

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Square Root (r)
* Decimal Number Support
* Division-by-Zero Protection
* Negative Square Root Validation
* Continuous Calculation Loop
* Exit Command

## Concepts Practiced

* Classes and Objects
* Member Functions
* Switch Statements
* Loops (`do-while`)
* Conditional Logic
* User Input and Output
* Function Parameters
* Error Handling
* Standard Math Library (`cmath`)

## How It Works

1. Enter the first number.
2. Select an operation:

   * `+` Addition
   * `-` Subtraction
   * `*` Multiplication
   * `/` Division
   * `r` Square Root
   * `e` Exit
3. Enter the second number if required.
4. The calculator displays the result.
5. Continue calculating or exit the program.
